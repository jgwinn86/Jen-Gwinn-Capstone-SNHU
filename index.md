# Jen Gwinn – CS499 Computer Science Capstone E‑Portfolio

## Professional Summary
I began the Computer Science program at SNHU with the intention of learning how to code and becoming a data analyst. Throughout the course of the course of the program, I have become familiar with multiple languages including Python, Java, C++, SQL, and more. I have also learned about data cleaning and validation, as well as automated testing techniques all with security at the forefront. The concepts that have stood out to me the most relate to the software development lifecycle, and the process of considering the user experience and the requirements to support the desired experience and functionality.

As I have been completing tis program I have also been working full time in product management in the financial industry and have been able to apply what I have been learning to improve my skills communicating with stakeholders and gaining the buy-in needed to take something from a concept or idea to an executed deliverable.

For this capstone, I am highlighting my skills in software engineering and design, data structures and algorithms, and databases – all with security in mind. My goal was to develop one artifact to showcase all of these skills while simulating a real-life scenario. I took a command-line based C++ program used to manage bids and developed a web-based dashboard that lives behind a log in. The dashboard itself shows software design and allows the user to see high-level monthly performance both table and graphs that adapt to multiple filters that highlight the enhancement to the data structures to identify historical trends. The dashboard also leads to a bid details page which lists out all the bids in detail with the same filters as the dashboard page as well as the ability to search for a specific bid by ID or keyword. To show database enhancement, some of the data is excluded based on the user type.
